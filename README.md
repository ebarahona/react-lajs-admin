## Getting Started

### 1. Installing packages

```
npm install
```

### 2. Starting the server
##### 1) Run webpack
```
node ./node_modules/webpack/bin/webpack.js
```
or you can install webpack globally so as to use it without referencing this path.

##### 2) Start the server
```
node .
```

## Additional commands
##### Linting files
```
npm run lint
```
